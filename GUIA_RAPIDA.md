# 🚀 Guía Rápida - Tu Mercadito

## ⚡ Inicio Rápido

### 1. Instalar Dependencias
```bash
cd e:\mypime\tu-mercadito
npm install
```

### 2. Ejecutar Proyecto
```bash
npm run dev
```

### 3. Abrir en Navegador
```
http://localhost:3000
```

## 📱 Vistas Implementadas

### Autenticación
- **/** - Pantalla Welcome (inicio)
- **/login** - Iniciar sesión
- **/signup** - Registro

### Navegación Principal
- **/home** - Inicio con productos
- **/categories** - Categorías
- **/search** - Búsqueda
- **/map** - Mapa de tiendas
- **/profile** - Perfil de usuario

### Detalles
- **/product/:id** - Detalle de producto
- **/store/:id** - Perfil de tienda

## 🎨 Características del Diseño

- ✅ Diseño minimalista azul y blanco
- ✅ Mobile-first responsive
- ✅ Navegación inferior con 5 opciones
- ✅ Formas curvas y bordes redondeados
- ✅ Grid de productos
- ✅ Formularios estilizados

## 📦 Tecnologías

- Vue 3 + Composition API
- Vite (build tool)
- TailwindCSS (estilos)
- Vue Router (navegación)
- Lucide Icons (iconos)

## 🔧 Comandos Útiles

```bash
# Desarrollo
npm run dev

# Build producción
npm run build

# Preview producción
npm run preview
```
